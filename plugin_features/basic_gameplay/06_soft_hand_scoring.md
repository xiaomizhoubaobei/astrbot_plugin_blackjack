# 软牌计分功能

## 功能概述
明确处理A作为1或11的计算规则，区分硬牌和软牌的概念。

## 功能细节
- 识别包含A的软牌组合（A可作为1或11）
- 明确显示当前最优点数计算方式
- 在策略提示中考虑软牌特殊情况
- 支持多张A的复杂计算

## 技术实现要点
- 实现软牌点数的准确计算算法
- 更新显示逻辑以区分硬牌和软牌
- 修改AI决策逻辑以考虑软牌
- 调整策略提示系统的计算

## 用户体验
- 清晰标识手牌中的A值选择（软17、硬17等）
- 在界面上直观显示当前点数计算
- 提供软牌特殊情况下的策略建议

## 预期收益
让玩家更好地理解21点的复杂计分规则，提升游戏的真实感。
