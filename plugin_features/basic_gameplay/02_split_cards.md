# 分牌功能

## 功能概述
当玩家初始两张牌点数相同时，可以选择分牌成两副独立的手牌进行游戏。

## 功能细节
- 初始两张牌点数相同（如两个8或两张Q）时提供分牌选项
- 分牌后每手牌分别进行游戏，需追加相同注额
- 可限制分牌次数（如最多分3手）
- A分牌有特殊规则（通常只能再摸一张牌）

## 技术实现要点
- 修改手牌管理逻辑以支持一位玩家多手牌
- 更新点数计算算法以处理多手牌
- 调整下注管理，确保分牌后追加注额正确
- 实现A分牌的特殊处理规则

## 用户体验
- 当可分牌时明确提示玩家此选项
- 直观显示分牌后的多手牌状态
- 明确标识当前正在操作的是哪一手牌

## 预期收益
提供更多策略选择，增加游戏深度，提升玩家决策的复杂性和乐趣。
