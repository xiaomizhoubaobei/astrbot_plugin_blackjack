# 双倍下注功能

## 功能概述
在特定情况下允许玩家加倍下注并只拿一张牌，适用于玩家认为下一张牌会使其获胜的情况。

## 功能细节
- 仅限初始两张牌时可执行双倍下注
- 下注金额必须等于原注额
- 执行后立即发一张牌并结束该手牌游戏
- 通常在硬9、10、11点时推荐使用

## 技术实现要点
- 增加双倍下注的游戏状态
- 实现双倍下注的合法性检查
- 调整结算逻辑以处理双倍下注的结果
- 修改决策流程，双倍后直接进入结算阶段

## 用户体验
- 在适合双倍下注时突出显示此选项
- 明确显示双倍下注的后果和收益
- 防止在不符合规则时误触双倍下注

## 预期收益
提供更多策略选择，增加高风险高回报的玩法，提升游戏刺激感。
