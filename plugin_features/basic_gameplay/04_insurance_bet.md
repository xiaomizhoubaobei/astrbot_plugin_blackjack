# 保险投注功能

## 功能概述
当庄家明牌是A时，玩家可以下保险注，防范庄家黑杰克的风险。

## 功能细节
- 庄家明牌为A时提供保险选项
- 保险注额为原注额的一半
- 若庄家确实是黑杰克，则保险注以2:1赔率赔付
- 若庄家不是黑杰克，则保险注失去

## 技术实现要点
- 增加保险投注的游戏状态判断
- 实现保险注的管理及结算
- 修改游戏流程，在庄家确认黑杰克前处理保险
- 调整庄家开牌逻辑以支持保险处理

## 用户体验
- 当庄家明牌为A时清晰提示保险选项
- 解释保险机制的运作原理
- 显示保险后的预期收益情况

## 预期收益
增加游戏策略元素，让玩家可以根据概率做出风险对冲决策。
